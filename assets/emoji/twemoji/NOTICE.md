These PNGs are used as the fallback emoji renderer on platforms without
Apple Color Emoji (e.g. Windows). On macOS, the app instead renders emoji
using the real system-installed Apple Color Emoji font directly — no Apple
font or graphics are bundled anywhere in this project.
